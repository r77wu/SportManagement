##  Setup
Config a MongoDB
```bash
DATABASE=<your MongoDB database>
DATABASE_PASSWORD=<your MongoDB password>
```
#### Backend 
```bash
npm install
npm run dev
```
#### Frontend 
```bash
npm install
npm start
```
